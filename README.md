课题要求：
在网页上制作一个table，table内容是：										
不能大于8个字节的字符串输入框 
必须验证为邮箱格式的邮箱输入框 
必须包含大小写与特殊符号的密码输入框 
必须将输入数字转写为汉字数字的年龄输入框										
必须按yyyy-mm-dd格式输入的日期输入框										
										
当右键点击该table时，在被点击的地方出现一个新建菜单，同时屏蔽网页原有的右键点击菜单（点击表格以外的部分还是出现原有的右键菜单，不能出现新建的菜单）。
新建菜单内容是两个选项：行追加，行删除			
							
行追加的作用是：在被点击的列后方追加一个新的行										
行删除的作用是：删除被选中的行							
追加删除功能与数据库联动 如果在页面上被删除，数据库中也会删除对应数据										
如果行追加，则每个输入框都需要前后端两次验证，前端验证一次是否符合格式，后端验证一次是否符合格式
